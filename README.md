# Node.js con MongoDB

Proyecto de curso práctico de Node.js implementando MongoDB como base de datos.

## Requisitos

- Node.js
- MongoDB

## Instalación

1. Clona el repositorio
   git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git

2. Instala las dependencias
   npm install

3. Crea un archivo .env con tus variables de entorno
   MONGO_URI=tu_conexion_de_mongodb

4. Inicia el servidor en modo desarrollo
   npm run dev

## Tecnologías

- Node.js
- Express
- MongoDB
- Dotenv
- Debug
- Nodemon
- Http-errors
- Excel4node
